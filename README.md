# EasyConvert Web

Ein einfaches Webtool zum Konvertieren von Dateien direkt im Browser.

🌐 **Webseite:** (https://breckztv-converter.netlify.app/)

## Features
- HTML + JS only
- Browser-basiert, keine Installation nötig
- Einfache Bedienung

## Nutzung
Einfach den Link aufrufen und Dateien hochladen / konvertieren.
